## 1. CLAT Quant Strategy
> *Incorporating Physics Wallah's High-Speed DI Strategy.*

CLAT Quantitative Techniques is **NOT a pure math section**. It is a **Data Interpretation** section:

- All questions are based on **graphs, tables, pie charts, or bar charts**
- You are tested on your ability to **read and extract data**, not compute complex equations
- Maximum calculation required: **two-step arithmetic**, ratios, and percentages

### The "Mental Math First" Rule
- **Don't use pen/paper immediately**. Read the table/chart entirely first. 
- Use the **Approximation Method**: If the options are wide apart (e.g., 20%, 35%, 50%), round off numbers (e.g., treat 18,943 as 19,000) to find the answer in 10 seconds.
- Never calculate absolute values if the question asks for a ratio or percentage. The bases often cancel out.

### Time Allocation
- 8 marks × ~1.5 minutes = **~12 minutes** total
- Read the data set once carefully (2 min) → answer questions (1 min each)

### Common Mistakes
- ❌ Doing complex calculations when the answer is visible by inspection
- ❌ Missing units (crores vs. lakhs, % vs. absolute numbers)
- ❌ Comparing numbers from different rows without checking the base year

## 2. Core Concepts — Quick Reference

### Percentage Change
```
% Change = ((New Value - Old Value) / Old Value) × 100
```

### Ratio & Proportion
- If A:B = 3:5 and total = 800, then A = 300, B = 500
- Cross-multiplication: A/B = C/D → A×D = B×C

### Average
```
Average = Sum of all values / Number of values
```

### Simple Interest vs. Compound Interest
```
SI = (P × R × T) / 100
CI = P × (1 + R/100)^T - P
```
> *CLAT rarely tests CI; SI more common in passage-based data*

### Fraction → Percentage Quick Reference

| Fraction | Percentage |
|---------|-----------|
| 1/2 | 50% |
| 1/3 | 33.33% |
| 1/4 | 25% |
| 1/5 | 20% |
| 1/8 | 12.5% |
| 2/3 | 66.67% |
| 3/4 | 75% |

## 5. Data Set 3: India's GDP & Sectoral Growth

### Pie Chart Data: GDP Composition by Sector (FY2025-26)

| Sector | % Share of GDP | Growth Rate FY26 |
|--------|---------------|-----------------|
| Services | 54% | 7.8% |
| Industry | 28% | 5.9% |
| Agriculture | 18% | 4.1% |

### GDP at Current Prices: ₹326 lakh crore (FY26 estimate)

### Practice Questions

**Q1.** What is the absolute value of the Services sector in GDP?
> 54% × 326 = **₹176.04 lakh crore**

**Q2.** If Agriculture grows at 4.1% in FY26, what will its absolute size be (starting from 18% of ₹305 lakh crore in FY25)?
> FY25 Agriculture = 18% × 305 = ₹54.9 LC; FY26 = 54.9 × 1.041 = **₹57.15 lakh crore**

**Q3.** Services grew at 7.8%. If this continues for 2 more years, what is the % growth over 2 years?
> Approx: 7.8 + 7.8 + (7.8×7.8/100) ≈ **16.2%** (using simple compounding approximation)

## 7. Formula Sheet

| Formula | Use |
|---------|-----|
| `% of X = (X/Total) × 100` | Finding share |
| `% Change = ((New-Old)/Old) × 100` | Growth rate |
| `Part = (% / 100) × Whole` | Finding absolute value from % |
| `Ratio A:B → A = Total × (a/(a+b))` | Distributing by ratio |
| `Average of n numbers = Sum / n` | Central tendency |
| `If A > B by x%, B < A by x/(100+x) × 100%` | Comparative percentage trap |
