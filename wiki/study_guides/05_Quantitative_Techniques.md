# 📊 CLAT Quantitative Techniques — Complete Study Guide

> **Section Weight**: 10–14 questions | ~8 marks | **10–12 minutes**  
> **Core Skill**: Class 10-level maths inside data sets, caselets, charts, and tables — **Data Interpretation is 80% of this section**.

---

## How CLAT Tests Quantitative Techniques

| Reality | Implication |
|---------|-------------|
| NOT a pure maths section | No complex algebra, trigonometry, or calculus |
| ALL questions are data-driven | Tables, bar graphs, pie charts, line graphs, caselets |
| Maximum calculation = 2-step arithmetic | Percentages, ratios, averages |
| Speed > precision | Approximation wins when options are spread apart |

### Time Strategy

- **Total time**: 10–12 minutes for entire section
- **Read data set once** (2 min) → **Answer questions** (~1 min each)
- **Skip and return** if a question takes >90 seconds

## PART A: DATA INTERPRETATION FORMATS

### A1. Caselets (Text Passages with Numbers)

**Example Caselet**:
> "Company X's revenue was ₹500 Cr in 2022, growing 20% in 2023. In 2024, revenue grew by half the 2023 growth rate. 40% of 2024 revenue came from exports."

**Extract first**:
- 2022: ₹500 Cr
- 2023: 500 × 1.20 = ₹600 Cr
- 2024 growth: 10% (half of 20%) → 600 × 1.10 = ₹660 Cr
- Exports 2024: 40% × 660 = ₹264 Cr

**Rule**: Underline every number; build a mini-table before answering.

### A2. Tables

**Reading Strategy**:
1. Read **row and column headers** first
2. Note **units** (₹ Cr vs ₹ Lakh, %, absolute numbers)
3. Note **base year** for comparisons
4. Look for **totals row/column**

### A3. Bar Graphs

- Compare **heights** for absolute values
- Compare **relative heights** for ratios
- **Stacked bars** = components of a total
- **Grouped bars** = comparison across categories/years

### A4. Line Graphs

- **Slope** = rate of change
- **Steep upward** = rapid growth
- **Intersection** = two values equal at that point
- **Plateau** = no change over period

### A5. Pie Charts

- Each slice = **percentage of whole**
- To get absolute value: **Percentage × Total**
- Compare slices visually for quick ratios
- **Multiple pie charts** = compare same category across charts

**Example**: Total budget ₹1000 Cr. Education = 25% → ₹250 Cr.

### A6. Multi-Layered Tables

- May have **sub-categories** within rows
- Check if percentages are **of total** or **of sub-category**
- Common trap: mixing percentage bases

## PART B: CORE ARITHMETIC

### B1. Percentages

**Basic Formulas**:
```
Percentage = (Part / Whole) × 100
Part = (Percentage / 100) × Whole
```

**Increase/Decrease**:
```
% Change = ((New - Old) / Old) × 100
New Value = Old × (1 + r/100)   [for increase]
New Value = Old × (1 - r/100)   [for decrease]
```

**Successive Changes**:
```
Net change = a + b + (a×b)/100   [for +a% then +b%]
Example: +10% then +20% = 10 + 20 + 2 = 32% net increase
```

**Fraction ↔ Percentage (Memorise)**:

| Fraction | % | Fraction | % |
|---------|---|---------|---|
| 1/2 | 50% | 1/8 | 12.5% |
| 1/3 | 33.33% | 3/8 | 37.5% |
| 1/4 | 25% | 5/8 | 62.5% |
| 1/5 | 20% | 7/8 | 87.5% |
| 1/6 | 16.67% | 1/10 | 10% |

### B2. Profit, Loss & Discount

```
Profit = SP - CP
Profit % = (Profit / CP) × 100
Loss = CP - SP
Discount = Marked Price - SP
Discount % = (Discount / MP) × 100
```

**Successive Discounts**: 10% + 20% ≠ 30%
```
Net = 100 × 0.90 × 0.80 = 72 → 28% total discount
```

**False Weight / Adulteration**:
```
Actual CP per unit = (Given CP × True weight) / False weight
```

### B3. Ratios & Proportions

```
If A:B = 3:5 and total = 800
A = (3/8) × 800 = 300
B = (5/8) × 800 = 500
```

**Partnership**:
- Simple: Profit shared in ratio of capital
- With time: Ratio = Capital₁ × Time₁ : Capital₂ × Time₂

**Componendo-Dividendo** (rarely needed but useful):
```
If a/b = c/d, then (a+b)/(a-b) = (c+d)/(c-d)
```

### B4. Averages

```
Simple Average = Sum of values / Number of values
```

**Weighted Average**:
```
Weighted Avg = (w₁×x₁ + w₂×x₂ + ...) / (w₁ + w₂ + ...)
```

**Average Speed** (NOT simple average of speeds):
```
Average Speed = Total Distance / Total Time
NOT (v₁ + v₂) / 2 unless distances equal
```

**Change in Average when one value added**:
```
New Avg = (Old Sum + New Value) / (n + 1)
```

### B5. Mixtures & Alligations

**Alligation Rule**:
```
(Cheaper quantity) / (Dearer quantity) = (Dearer price - Mean) / (Mean - Cheaper price)
```

**Example**: Mix ₹20/kg rice with ₹50/kg rice to get ₹35/kg mixture.
```
Qty ratio = (50-35)/(35-20) = 15/15 = 1:1
```

**Replacement Formula** (replacing x litres of y% solution with pure solution n times):
```
Final concentration = y × (1 - x/V)ⁿ
```

### B6. Simple Interest & Compound Interest

```
SI = (P × R × T) / 100
Amount = P + SI
CI = P(1 + R/100)ᵀ - P
```

**Difference between CI and SI (2 years)**:
```
CI - SI = P × (R/100)²
```

**CLAT Note**: SI is more common in passages; CI appears occasionally.

### B7. Time, Speed & Distance

```
Speed = Distance / Time
Distance = Speed × Time
Time = Distance / Speed
```

**Relative Speed**:
- Same direction: |S₁ - S₂|
- Opposite direction: S₁ + S₂

**Trains**:
```
Time to cross pole = Length of train / Speed
Time to cross platform = (Train + Platform length) / Speed
```

**Boats & Streams**:
```
Downstream speed = Boat + Stream
Upstream speed = Boat - Stream
Boat speed = (Down + Up) / 2
Stream speed = (Down - Up) / 2
```

### B8. Time & Work

```
Work = Rate × Time
If A completes in 6 days: Rate = 1/6 per day
Combined rate = 1/A + 1/B
Combined time = 1 / (1/A + 1/B)
```

**Efficiency ratio**: If A:B efficiency = 3:2, A does 3/5 of work, B does 2/5.

### B9. Basic Mensuration

**2D Shapes**:

| Shape | Area | Perimeter |
|-------|------|-----------|
| Rectangle | l × b | 2(l + b) |
| Square | a² | 4a |
| Triangle | ½ × b × h | a + b + c |
| Circle | πr² | 2πr |

**3D Shapes**:

| Shape | Volume | Surface Area |
|-------|--------|-------------|
| Cuboid | l × b × h | 2(lb + bh + hl) |
| Cube | a³ | 6a² |
| Cylinder | πr²h | 2πr(r + h) |
| Sphere | (4/3)πr³ | 4πr² |

**CLAT Tip**: π ≈ 22/7 or 3.14; use approximation when options allow.

## PART C: PRACTICE DATA SETS

### Data Set 1: Union Budget Expenditure (₹ Lakh Crore)

| Department | FY25 (RE) | FY26 (BE) |
|------------|-----------|-----------|
| Defence | 6.22 | 6.81 |
| Interest Payments | 11.62 | 12.36 |
| Rural Development | 2.66 | 2.98 |
| Agriculture | 1.51 | 1.71 |
| Education | 1.25 | 1.28 |
| Health | 0.90 | 0.99 |
| Capital Expenditure | 10.18 | 11.21 |
| **Total** | **47.16** | **50.65** |

**Q1.** Highest % increase from FY25 to FY26?
> Agriculture: (1.71-1.51)/1.51 × 100 ≈ **13.2%**

**Q2.** Interest payments as % of total FY26?
> 12.36/50.65 × 100 ≈ **24.4%**

**Q3.** If Education must be 3% of total FY26 (50.65), required allocation?
> 3% × 50.65 = 1.52; current = 1.28; **increase ₹0.24 lakh crore**

### Data Set 2: Pie Chart — Book Publishing Costs

| Component | Share |
|-----------|-------|
| Printing | 18% |
| Paper | 16% |
| Editing | 36% |
| Royalty | 12% |
| Miscellaneous | 10% |
| Proof-reading | 8% |

Total cost = ₹500 per book.

**Q4.** Cost of editing per book?
> 36% × 500 = **₹180**

**Q5.** Printing + Paper combined as fraction of total?
> 18% + 16% = 34% = **17/50**

**Q6.** If royalty increases by 50%, new royalty cost?
> 12% × 1.5 = 18%; 18% × 500 = **₹90** (was ₹60)

### Data Set 3: Caselet — Company Revenue

> "TechCorp's Q1 revenue was ₹200 Cr. Q2 saw 25% growth. Q3 revenue fell by 20% from Q2. Export share was 30% in Q1, rising by 5 percentage points each quarter."

**Extract**:
- Q1: ₹200 Cr (Export: 30% = ₹60 Cr)
- Q2: 200 × 1.25 = ₹250 Cr (Export: 35% = ₹87.5 Cr)
- Q3: 250 × 0.80 = ₹200 Cr (Export: 40% = ₹80 Cr)

**Q7.** Q2 domestic revenue?
> 250 × 65% = **₹162.5 Cr**

**Q8.** Overall growth Q1 to Q3?
> (200-200)/200 = **0%** (back to start)

### Data Set 4: Time-Speed (Caselet)

> "A train 180m long crosses a platform 120m long in 15 seconds. Another train 150m long crosses a pole in 10 seconds."

**Train 1**: Total distance = 180 + 120 = 300m in 15 sec → Speed = 300/15 = **20 m/s**

**Train 2**: Distance = 150m in 10 sec → Speed = **15 m/s**

**Q9.** Time for Train 1 to cross Train 2 (opposite directions)?
> Relative speed = 20 + 15 = 35 m/s; Total distance = 180 + 150 = 330m; Time = 330/35 ≈ **9.4 seconds**

### Data Set 5: Mixtures

> "A merchant has 20 litres of milk at ₹40/litre. He adds 5 litres of water (free). What is the new price per litre if he wants 25% profit on cost?"

**New CP per litre** = (20 × 40) / 25 = 800/25 = **₹32/litre**
**SP for 25% profit** = 32 × 1.25 = **₹40/litre**

## PART D: APPROXIMATION TRICKS

| Situation | Trick |
|-----------|-------|
| Options differ by >5% | Round numbers (18,943 → 19,000) |
| Ratio asked | Cancel common factors before calculating |
| Percentage of total | Use fractions (25% = 1/4) |
| Compare two fractions | Cross-multiply |
| Large multiplication | Break into easy parts (998 × 5 = 1000×5 - 2×5) |

**Example**: 19.4% of 503 ≈ 20% of 500 = 100 (actual ≈ 97.5 — close enough if options are 90, 97, 105, 120)

## PART E: COMMON MISTAKES

| Mistake | Fix |
|---------|-----|
| Wrong units (Cr vs Lakh) | Circle units before calculating |
| % increase vs % of total | "Increase BY 20%" ≠ "20% of total" |
| Average speed error | Use total D / total T |
| Pie chart: % of wrong total | Confirm which total the % refers to |
| Over-calculation | Check if answer visible by inspection |
| Successive % addition | Use multiplicative formula |
