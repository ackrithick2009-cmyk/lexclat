# 🧠 CLAT Logical Reasoning — Complete Study Guide

> **Section Weight**: 22–26 questions | ~22 marks | **25 minutes**  
> **Core Skill**: Critical (verbal) reasoning dominates — analyse arguments, assumptions, flaws, and inferences from passages.

## How CLAT Tests Logical Reasoning

Post-2020 CLAT format: **Every question is passage-based** — no standalone puzzle sets (though short analytical sets appear embedded in passages).

| Category | Weight | Format |
|----------|--------|--------|
| **Critical Reasoning** | ~70% | 250–450 word passage + 4–5 questions |
| **Analytical Reasoning** | ~30% | Short sets: syllogisms, seating, blood relations, coding |

## PART A: CRITICAL REASONING (HIGH PRIORITY)

### A1. Arguments, Premises & Conclusions

**The PAC Framework**:

| Component | Definition | Signal Words |
|-----------|-----------|--------------|
| **Premise (P)** | Evidence supporting the claim | "Because," "Since," "Given that," "Studies show" |
| **Assumption (A)** | Unstated bridge from premise to conclusion | (Not stated — you identify it) |
| **Conclusion (C)** | Main claim the author wants you to accept | "Therefore," "Thus," "Hence," "It follows that," "Must" |

**Example**:
> "Because organised crime syndicates intimidate witnesses when accused are on bail, **mandatory pre-trial detention is justified**."

- **Premise**: Witness intimidation when accused on bail
- **Conclusion**: Mandatory pre-trial detention is justified
- **Hidden Assumption**: Pre-trial detention effectively prevents witness intimidation better than alternatives

### A2. Strengthening Arguments

**To STRENGTHEN**: Find an option that:
- Supports a premise with new evidence
- States the hidden assumption explicitly
- Eliminates an alternative explanation
- Shows cause-effect link is real (not just correlation)

**Example — STRENGTHENS**:
> Studies show witness intimidation is 3× more likely in organised crime cases when accused have bail.

### A3. Weakening Arguments

**To WEAKEN**: Find an option that:
- Breaks the link between premise and conclusion
- Provides an alternative cause
- Shows the premise is false or overstated
- Demonstrates the proposed solution doesn't work

**Example — WEAKENS**:
> Courts have held that pre-trial detention without individual risk assessment violates Article 21; case-specific bail conditions are equally effective.

### A4. Spotting Assumptions

**The Negation Test**:
1. Take each option
2. Negate it (make it the opposite)
3. If negating it **destroys the argument** → it's a necessary assumption

**Example**:
> Argument: "This law school produced the most Supreme Court judges; therefore it has the best legal pedagogy."
>
> Assumption: "Becoming a Supreme Court judge reflects the quality of legal training."
>
> Negate: "Becoming an SC judge does NOT reflect training quality" → Argument collapses. ✅

**Assumption vs Inference**:

| | Assumption | Inference |
|--|-----------|-----------|
| Direction | Hidden **premise** behind argument | Logical **result** from stated facts |
| Test | Negation destroys argument | Must follow from evidence |
| Stated? | Never stated | Implicitly supported |

### A5. Logical Flaws

| Flaw | Definition | Example |
|------|-----------|---------|
| **Ad Hominem** | Attack person, not argument | "His argument is wrong because he's a criminal" |
| **False Dilemma** | Only two options when more exist | "Either ban social media or accept misinformation" |
| **Slippery Slope** | One step → extreme chain | "Assisted dying → killing healthy people" |
| **Appeal to Authority** | Authority alone as proof | "Eminent lawyers support it, so it's constitutional" |
| **Circular Reasoning** | Conclusion in premise | "It's wrong because it's illegal; it's illegal because it's wrong" |
| **Hasty Generalisation** | Small sample → broad claim | "One corrupt lawyer → all lawyers are corrupt" |
| **Post Hoc** | A before B → A caused B | "Crime fell after new law → law caused it" |
| **Straw Man** | Misrepresent opponent's argument | Attacking a weaker version |
| **Equivocation** | Same word, different meanings | "Banks are financial institutions; river banks hold water; therefore..." |

### A6. Inferences & Paradoxes

**Valid Inference Rules**:
- Must be **supported by the passage**
- Prefer **conservative** claims ("may," "suggest," "likely")
- Must NOT require outside knowledge

**Paradox Resolution**: Apparent contradiction → find the missing variable that explains both facts.

> "Country X has the highest tax rates but the lowest tax revenue collection."
>
> **Resolution**: High rates → evasion; poor enforcement; large informal economy.

### A7. Cause & Effect

| Relationship | Test |
|-------------|------|
| **Causation** | A directly produces B |
| **Correlation** | A and B occur together — no proven causal link |
| **Reverse Causation** | B actually causes A (not A causes B) |
| **Third Variable** | C causes both A and B |

**CLAT Trap**: Passage states two facts occurring together → question asks if one caused the other. Check if passage **establishes** causation or merely correlation.

## PART B: ANALYTICAL REASONING

### B1. Syllogisms

**Structure**: Two premises → Conclusion

**Standard Form**:
- All A are B
- All B are C
- ∴ All A are C ✅

**Valid Patterns** (memorise these):

| Premise 1 | Premise 2 | Valid Conclusion |
|-----------|-----------|-----------------|
| All A are B | All B are C | All A are C |
| No A is B | All C are A | No C is B |
| Some A are B | All B are C | Some A are C |
| All A are B | Some C are A | Some C are B |

**Invalid Patterns** (common traps):
- ❌ All A are B + All C are B → All A are C (undistributed middle)
- ❌ Some A are B + Some B are C → Some A are C (not necessarily)

**Venn Diagram Method**:
1. Draw circles for each category
2. Shade areas that are **empty**
3. Mark areas where something **definitely exists**
4. Read conclusion from diagram

**"Only" and "Some" Quantifiers**:
- "Only A are B" = All B are A (reversed!)
- "Some A are not B" = At least one A is outside B
- "None but A are B" = All B are A

### B2. Linear Seating Arrangements

**Steps**:
1. Fix one person as reference point
2. Apply direct constraints first ("A sits to immediate left of B")
3. Apply negative constraints ("C does not sit next to D")
4. Fill remaining gaps

**Example**: 5 people A, B, C, D, E sit in a row.
- A is not at either end
- B sits exactly in the middle
- C sits to the immediate right of A

**Solution**: _ _ B _ _ → A not at end → A in pos 2 or 4. C right of A → A=2, C=3. B=3? No, B is middle (pos 3). So B=3, A=2, C=4. Pos 1 and 5: D, E. **Order: D-A-B-C-E or E-A-B-C-D**

### B3. Circular Seating Arrangements

- Fix one person at 12 o'clock
- "Left" and "right" refer to **clockwise/anticlockwise from that person's perspective**
- n people → n positions; no "ends"

### B4. Blood Relations (Family Trees)

**Key Symbols**:
- + = married
- | = parent-child (vertical)
- — = siblings (horizontal)

**Quick Rules**:
| Term | Meaning |
|------|---------|
| Maternal uncle | Mother's brother |
| Paternal aunt | Father's sister |
| Cousin | Child of parent's sibling |
| Nephew/Niece | Sibling's child |
| In-law | Spouse's family |

**Technique**: Start from the known person; work backwards/forwards one relation at a time.

**Example**: "Pointing to a man, a woman said, 'His mother is the only daughter of my mother.' How is the woman related to the man?"
- "Only daughter of my mother" = the woman herself
- Man's mother = the woman
- Woman is man's **mother**

### B5. Coding-Decoding

**Types**:

| Type | Pattern |
|------|---------|
| **Letter shifting** | Each letter +1 (CAT → DBU) |
| **Reverse alphabet** | A=Z, B=Y (CAT → XZG) |
| **Number substitution** | A=1, B=2 (CAT → 3-1-20) |
| **Word substitution** | "sky" = "book," "blue sky" = ? → "blue book" |

**Strategy**: Find the pattern in the given pair first; apply to the question.

### B6. Analogies

**Format**: A : B :: C : ?

**Relationship Types**:

| Type | Example |
|------|---------|
| **Synonym** | Happy : Joyful :: Sad : ? (Gloomy) |
| **Antonym** | Hot : Cold :: Day : ? (Night) |
| **Part-Whole** | Page : Book :: Room : ? (House) |
| **Cause-Effect** | Fire : Burn :: Rain : ? (Flood) |
| **Function** | Knife : Cut :: Pen : ? (Write) |
| **Degree** | Warm : Hot :: Dislike : ? (Hate) |
| **Worker-Tool** | Carpenter : Saw :: Surgeon : ? (Scalpel) |

**Rule**: The relationship between A and B must be **identical** to the relationship between C and the answer.

## PART C: PRACTICE SETS

### Set 1: Bail & Pre-Trial Detention (Critical Reasoning)

> The Supreme Court has held that bail is the rule and jail is the exception in non-bailable offence cases. The state must demonstrate that the accused poses a flight risk, risk of evidence tampering, or threat to witnesses to justify pre-trial detention. Economic offences, however, have been treated as a special category where the gravity of the offence is itself a relevant consideration.

**Q1.** Rajan, accused of fraud involving ₹50,000, seeks bail. Prosecution argues detention because it's an economic crime. Which most strengthens Rajan's application?
- (A) Rajan has deep community roots and no prior record
- (B) Prosecution hasn't shown flight risk or witness threat
- (C) The amount is too small for "serious" economic offence
- (D) Both (A) and (B)

**Answer: (D)**

**Q2.** Which is an assumption underlying the passage's general rule?
- (A) All accused persons are innocent until proven guilty
- (B) Pre-trial detention is never justified
- (C) Economic offences are always bailable
- (D) The state always opposes bail

**Answer: (A)** — "Bail is the rule" presumes presumption of innocence.

### Set 2: Free Speech (Critical Reasoning)

> Article 19(1)(a) is subject to reasonable restrictions under Article 19(2), including public order. A restriction is 'reasonable' only if proportionate — the harm prevented must outweigh the right curtailed. Speech creating 'clear and present danger' of imminent unlawful action may be restricted, but mere offensiveness does not meet this threshold.

**Q3.** A comedian's satirical video is blocked for being "offensive." Best-supported conclusion?
- (A) Action is constitutional — protects public order
- (B) Mere offensiveness doesn't justify restriction; block likely unconstitutional
- (C) Valid under Article 19(2) indecent content
- (D) Comedian needed government clearance

**Answer: (B)**

### Set 3: Syllogism

**Q4.** Statements: All lawyers are argumentative. Some argumentative people are politicians. Conclusions: I. Some lawyers are politicians. II. Some politicians are argumentative.
- (A) Only I follows
- (B) Only II follows
- (C) Both follow
- (D) Neither follows

**Answer: (B)** — II follows (some argumentative = politicians). I doesn't necessarily follow (lawyers ⊆ argumentative, but politicians may not overlap with lawyers).

### Set 4: Blood Relations

**Q5.** A is B's sister. C is B's mother. D is C's father. E is D's daughter. How is A related to E?
- (A) Aunt
- (B) Sister
- (C) Niece
- (D) Cousin

**Answer: (A)** — E is D's daughter = C's sister (since D is C's father). A is C's child. E is A's aunt... Wait, let me recalculate.
- C is B's mother, A is B's sister → A and B are siblings, C is their mother
- D is C's father → D is grandparent of A and B
- E is D's daughter → E is C's sister → E is A's **aunt**

**Answer: (A) Aunt**

## PART D: EXAM STRATEGY

### Time Allocation

| Question Type | Time per Q |
|--------------|-----------|
| Critical Reasoning (passage) | 1–1.5 min |
| Syllogism | 45 sec |
| Seating/Blood Relations | 1–1.5 min |
| Coding-Decoding | 45 sec |

### Trap Options to Eliminate First

1. **Irrelevant true facts** — true but don't affect the argument
2. **Scope shifts** — addresses different issue
3. **Extreme language** — "always," "never," "all," "none"
4. **Over-inference** — goes beyond what passage supports

### Closest Answer Logic

When two options seem correct:
1. Which is **more directly supported** by the passage?
2. Which is **narrower** (less extreme)?
3. Which addresses the **exact question asked** (strengthen vs weaken vs assumption)?
